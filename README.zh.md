# kaguya

**kaguya** 是一个用 Go 语言构建 AI Agent 应用的实验性项目。

## 谁是 Kaguya？

**Kaguya** 源自 **Kaguya-hime / 辉夜姬**。

在日本古典意象中，辉夜姬是月宫的公主：静谧、优雅、遥远而神秘。

在《龙族》中，**Kaguya** 也与日本分部的超级人工智能系统相关联——一个隐藏在组织背后的冷静、理性且强大的智能体。

本项目借用了这一形象。

`kaguya` 被设计为一个安静、理性、可控的 Agent 核心。

## 关于项目

`kaguya` 是一个 Go 原生项目，用于探索 AI Agent 的设计与实现。

项目聚焦于 Agent 的基本结构，包括模型抽象、提示词处理、工具调用和执行流程。

目前项目处于早期开发阶段，主要用于学习、实验和渐进式框架设计。
